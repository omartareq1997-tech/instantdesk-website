# InstantDesk homepage asset slots

Replace these placeholder slots with final media before launch:

- `hero-video.mp4` - cinematic homepage background video.
- `hero-image.webp` - full-width product or owner/client lifestyle visual.
- `customer-1.webp` - primary customer or industry image.
- `customer-2.webp` - secondary customer or industry image.
- `customer-3.webp` - tertiary customer or industry image.
- `instantdesk-logo.png` - real InstantDesk logo file used in the navbar and footer.

The homepage currently renders designed placeholder blocks and marks each slot with `data-asset-slot`.
